Sentiment
Analysis Project

Sentiment Analysis on 2024 News Articles regarding Taylor Swift's recent endorcement of Kamala Harris in the 2024 Election.

The objective of this project was to use sentiment analysis models to determine how news outlets felt about Taylor Swift endorsing Kamala Harris. All of our work was done using a number of Python libraries like: BeautifulSoup, NLTK, Spacy, Pandas, NewsAPI, and MatPlotLib. To collect data, the NewsAPI collects articles about Taylor Swift's endorsement. We created a list of URLs, and the BeautifulSoup package converts them to workable text. All relevent articles and articles that do not have paywalls are filtered and stored in a list. The preprocessing portion of the data occurs when articles are turned into NLP objects using Spacy. Using token labels, stop words and punctuation are removed from the remaining text. At this point, only lemmatized tokens remain. Our group used three Sentiment models to analyze the data collected on the news articles: TextBlob, Vader, and Bert. As shown on the graph above, our results determine that all 3 models show a positive reaction to Taylor Swift's endorsement of Kamala Harris.
